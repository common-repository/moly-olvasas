== Description ==

With Moly olvasas widget you can add your moly.hu reading list to your wordpress blog.

== Installation ==

1. Download the zip file and extract the contents.
2. Make sure you're running WordPress version 2.3 or better. It won't work with older versions.
3. Upload the 'moly-olvasas-widget' folder to your plugins directory (plugins/moly-olvasas-widget).
4. Activate the plugin through the 'plugins' page in WP.
5. See widget options to set your login name and the cover visibility.
